# Installation Support Policy (NRE)

This policy covers installation-only support to help customers run NRE successfully for the
first time. It does not include customization, feature development, or ongoing maintenance.

## Scope
- Environment setup guidance (Python, virtualenv, dependencies)
- Configuration help (.env, config.json)
- First successful run of `python nre_core.py`

## Out of Scope
- Custom persona design or prompt engineering
- Feature requests, refactors, or code changes
- Infrastructure troubleshooting outside the local environment
- Model usage costs or API billing issues

## Eligibility and Deposit
- Installation support is optional and requires a deposit before scheduling.
- The deposit is 50% of the agreed support fee.
- The remaining 50% is due after a successful first run.
- Deposits are non-refundable once scheduling begins.

## Support Window and Limits
- Valid for 30 days after delivery.
- Up to 3 incidents or 3 total hours, whichever comes first.
- Remote support only.

## Response Time
- Initial response within 2 business days.

## What You Must Provide
- A valid OpenAI API key.
- A machine that meets the runtime requirements.
- Remote access or logs as requested.

## Success Criteria
- NRE runs successfully with a sample prompt on the customer environment.
