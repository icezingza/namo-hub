# NaMo Memory API - Discovery Call Script

## Opening (2 minutes)
- Confirm role and current initiatives.
- Set agenda: problem, scope, timeline, budget, next steps.

## Qualification Questions
1. Which memory use case matters most: personalization, RAG caching, or agent memory?
2. What is your target launch window for a pilot?
3. Which cloud stack do you use today?
4. Who owns API contracts and data governance?
5. What KPIs define success (latency, MRR@k, error rate, retention)?
6. What is the current integration bottleneck?
7. What security or compliance constraints apply?

## Scope Confirmation
- Summarize the main use case and desired timeline.
- Confirm required deliverables and stakeholders.

## Close
- Propose Tier 1 or Tier 2 based on scope.
- Ask for approval to send proposal and request a deposit.

## Next Steps
- Send proposal within 24 hours.
- Schedule kickoff after approval and payment.
